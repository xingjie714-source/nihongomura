## 技術的な詳細

### データベース構造
- `profiles`テーブルにユーザー情報を保存
- Supabase Authと連携した認証システム

### 管理者ダッシュボードの機能
```javascript
// ユーザーデータ取得
await supabaseClient
    .from('profiles')
    .select('*')
    .order('created_at', { ascending: false });
```

### 登録フロー
1. ユーザーがフォームに入力
2. Supabase Authでユーザー登録
3. profilesテーブルにプロフィール情報を保存
4. 登録完了ページにリダイレクト
5. 確認メール送信（Supabase設定による）

## 注意事項

### Supabase設定
- メール確認機能を無効にしている場合、即座にログイン可能
- メール確認機能を有効にしている場合、確認メールのリンクをクリック後にログイン可能

### 管理者権限
- 管理者ダッシュボードにアクセスするには、profilesテーブルの`role`カラムが`admin`である必要があります
